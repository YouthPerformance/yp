# SEO Generate Skill

Generate SEO content pages directly from the Knowledge Graph.

## Command
`/seo-generate [cluster] [count]`

## Clusters Available
- `silent-basketball` - Silent/quiet training content (Adam voice, NeoBall CTA)
- `home-training` - Home workout content (Adam voice, NeoBall CTA)
- `ball-handling` - Handle development (Adam voice)
- `shooting` - Shooting form/technique (Adam voice)
- `barefoot` - Barefoot training (James voice)

## Output Location
`tools/machine-sprint/output/pages/[taskId].json`
